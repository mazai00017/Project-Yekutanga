Research resources with comments

Each resource below includes a short note explaining how it supports the website. These sources replace the more formal standards-document references so the research page fits the class project style better.

MDN Web Docs: HTTP - Hypertext Transfer Protocol

Used for the basic definition of HTTP, the client-server model, request and response concepts, methods, headers, status codes, and the overall role of HTTP in web development.

High Performance Browser Networking: Brief History of HTTP

Used to explain how HTTP grew from simple document retrieval into a performance-sensitive protocol that supports modern web applications and richer pages.

INNOQ: A Brief History of HTTP

Used for a more current historical overview that connects HTTP/1.1, HTTP/2, and HTTP/3 with the web's shift toward faster and more reliable communication.

U.S. Web Design System: HTTP/2 Performance Guide

Used for the performance section, especially the explanation that HTTP/2 changed how resources are downloaded and affected older optimization habits.

SecurityScorecard: What Is HTTPS and Why Is It Essential?

Used to connect HTTPS with cybersecurity, encryption in transit, protection from interception, and trust between users and websites.

Fortinet: What Is HTTPS?

Used for a clear security-focused explanation of how HTTPS encrypts data sent between users and websites and why it is safer than plain HTTP.

UpGuard: What Is HTTPS and Why Is It Important?

Used to support the discussion of HTTPS as the secure version of HTTP and to explain why encrypted browsing matters for privacy and risk reduction.

W3Schools: HTML Forms and JavaScript Form Validation

Used while building the quiz page, including form structure, input controls, buttons, and basic JavaScript validation ideas.

IT 3203 class examples and project material

Used to keep the project aligned with class expectations for separate HTML pages, external CSS, external JavaScript, navigation, forms, and GitHub Pages publishing.

GitHub Pages Documentation: Creating a GitHub Pages Site

Used for deployment guidance so the finished project can be published from the repository and opened with the public GitHub Pages URL.

Browser Developer Tools

Used for testing and debugging layout, mobile viewport behavior, navigation, file paths, console errors, and the quiz script during the Milestone 3 revision.
